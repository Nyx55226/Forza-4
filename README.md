# Forza-4

Project developed entirely in Java, implementing the classic Connect Four game with a graphical user interface. The program allows two players to compete by alternating moves on a vertical grid, updated in real time, with automatic turn management and win condition detection.


Progetto sviluppato interamente in Java, che implementa il classico gioco del Forza 4 con interfaccia grafica. Il programma consente a due giocatori di sfidarsi alternando le mosse su una griglia verticale, aggiornata in tempo reale, con gestione automatica dei turni e rilevamento delle condizioni di vittoria.
