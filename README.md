# Forza-4

Java implementation of the Connect Four game, featuring turn-based gameplay, move validation and automatic win condition detection.


Implementazione del gioco Forza 4 in Java, con gestione dei turni, validazione delle mosse e rilevamento automatico delle condizioni di vittoria.
